# kicad-library
## KiCad small custom library for work purposes

## KiCad pathes
Add to KiCad pathes this strings:
- KICAD_USER_SYMBOL_DIR: path-to-library-dir
- KICAD_USER_SYS3DMOD: path-to-packages3d-dir
- KICAD_USER_SYSMOD: path-to-modules-dir
- KICAD_USER_SYSMOD: path-to-modules-dir
